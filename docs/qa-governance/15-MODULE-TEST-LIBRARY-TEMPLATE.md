# Module Test Library Template

每個 Module Test Library 必須記錄 Module、Contract Version、Schema Version、Owner、Coverage Matrix、Positive／Negative／Permission／Tenant Isolation／Concurrency／Replay／Recovery／Reconciliation／Performance／Security Tests、Open Gaps、Approval Status。

## Point Skeleton

- Contract／Schema Version：Pending
- Coverage Matrix：Planned
- Concurrency／Ledger／Projection／Idempotency／Reverse Tests：Planned
- Approval Status：No

## Referral Skeleton

- Contract／Schema Version：Pending
- Relationship uniqueness、cycle、correction、Tenant Isolation Tests：Planned
- Approval Status：No

## Attendance Skeleton

- Contract／Schema Version：Pending
- Session scope、duplicate、permission、replay、reconciliation Tests：Planned
- Approval Status：No

以上僅為結構示例，不代表 Test Implemented、Executed、Passed 或 Verified。完整複製模板見 [Module Test Library Template](templates/MODULE-TEST-LIBRARY-TEMPLATE.md)。
