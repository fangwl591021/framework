# RC1 Known Limitations

以下限制在 RC1 建立時仍未解決，任何一項都不得被文件狀態隱含視為已完成：

1. Internal ID generator 尚未決策。
2. UTC time storage 與 D1 實際語意尚未驗證。
3. Physical Schema Proposal 尚未在 Local 或 Isolated D1 測試。
4. A01～A06 負向案例尚未執行。
5. Query plan、Index 效益與寫入成本尚未量測。
6. Durable Object 啟用門檻與 Hot Account 指標尚未決策。
7. Audit retention 尚未核准。
8. Idempotency retention 尚未核准。
9. Attribution Touch archive policy 尚未核准。
10. Shared／Tenant Group／Dedicated D1 topology 尚未決策。
11. Runtime Repository 尚未建立。
12. Security Review 與 Execution Approval 尚未完成。

限制的解除必須留下決策、測試證據或核准紀錄；不得只修改狀態文字。
